# Kanban Component Library

A production-grade Kanban Board View component library built with React, TypeScript, Tailwind CSS, and Storybook.

## Setup

This project uses Vite, React, TypeScript, Tailwind CSS, and Storybook.

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Run Storybook
npm run storybook

# Build Storybook
npm run build-storybook

# Build for production
npm run build
```
